# CPP05_09
Extended C++ and OOP concepts, this modules cover even bigger part of this beautiful language.


## CPP06

 - Examples of char literals: ’c’, ’a’, ...
    To make things simple, please note that non-displayable characters shouldn’t be used as
    nputs. If a conversion to char is not displayable, print an informative message.
 - Examples of int literals: 0, -42, 42...
 - Examples of float literals: 0.0f, -4.2f, 4.2f...
    You have to handle these pseudo-literals as well (you know, for science): -inff, +inff,
    and nanf.
 - Examples of double literals: 0.0, -4.2, 4.2...
    You have to handle these pseudo-literals as well (you know, for fun): -inf, +inf, and
    nan.
